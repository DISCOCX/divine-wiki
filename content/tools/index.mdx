# ✦ DIVINE TOOLS

[Quartz | Alround Tool to edit bins, textures and audio.](/wiki/tools/tex-thumbnail-provider)

:::tabs General Tools
::tab Free
[RitoBin | Converts `.bin` files to `.py` and back using `ritobin_cli`.](/wiki/tools/ritobin)
[Obsidian | Main tool to extract and browse League game files.](/wiki/tools/obsidian)
[CS-LoL Manager | Used to install mods and create mod packages (fantomes).](tools/cs-lol-manager)
[LtMAO | LtMAO-hai is a toolpack to help League modding.](/wiki/tools/ltmao)
:::

---

:::tabs 3D Modeling
::tab Free
[Blender | NOT RECOMMENDED / Free program to model, animate, rig, and edit 3D assets.](tools/blender)
::tab Paid
[Autodesk Maya | Industry-standard 3D software, used for rigging and UV mapping. Perfect for League modding.](tools/maya)
:::

---

:::tabs Texture Editing
::tab Free
[GIMP | Open-source image editor for `.dds` textures. Supports plugins.](https://www.gimp.org/downloads/)
[tex2dds | Converts `.tex` files to `.dds` and back / LtMAO recommended.](https://github.com/Morilli/Ritoddstex/releases)
::tab Paid
[Photoshop | Used to manually recolor and edit `.dds` textures with DXT5 compression.](/wiki/tools/photoshop)
:::

---

:::tabs VFX / Particle Editing
::tab Free
[Hacksaw | Recolors and edits `.bin` VFX files; supports palette-based editing.](/wiki/tools/Hacksaw)
:::

---

:::tabs Code
::tab Free
[Visual Studio Code | Used to view/edit `.py` particle scripts converted via RitoBin.](/wiki/tools/visual-studio-code)
:::

---

:::tabs Audio Editing
::tab Free
[Quartz | Has a .wpk editor / wem.](/wiki/tools/tex-thumbnail-provider)
[LtMAO | Recommended, can convert wem to wav and back.](/wiki/tools/ltmao)
[bnk-GUI | Extracts and rebuilds `.wem` files from `.bnk` soundbanks.](https://github.com/Morilli/bnk-extract-GUI/releases/tag/v1.5)
[Foobar2000 | Ltlmao better. Converts `.wem` files to `.wav` for audio editing.](https://www.foobar2000.org/)
[Audacity | Edits `.wav` audio files (cutting, layering, exporting).](https://www.audacityteam.org/)
[Wwise 2019 |  Required to make custom `.bnk` files. (advanced)](https://www.audiokinetic.com/en/download/)
:::
